## Overview
This project is an **IPL Dashboard** built using **Power BI** to analyze and visualize data from the **Indian Premier League (IPL)**. The dashboard provides insights into  IPL Overview over the seasons, Team Overview, Player overview, and other key metrics.

- **Team Performance Analysis**: Win/loss ratio, highest run scorers, best bowlers, etc.
- **Player Statistics**: Runs, wickets, strike rates, economy rates.
- **Match Insights**: Match results, top performers, venues, and more.
- **Season-wise Comparisons**: Trends across different IPL seasons.
- **Interactive Visuals**: Filters and slicers for better data exploration.
